# MedClaim AI

Intelligent Revenue Cycle and Claim Risk Management Platform designed to help healthcare facilities identify insurance claims that are likely to be denied before they are ever submitted.

## Features
- Predicts insurance claim approval or denial
- Machine learning-based risk prediction
- Claim correction recommendations
- Interactive Streamlit dashboard
- PostgreSQL database integration

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas
- PostgreSQL

## Installation

```bash
git clone https://github.com/Fanu489/MedClaim-AI.git
cd MedClaim-AI
pip install -r requirements.txt
streamlit run streamlit_app.py
```
